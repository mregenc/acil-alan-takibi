# Acil Alan Takip PWA

## Kurulum

1. Terminal açın.
2. Bu klasöre girin.
3. Aşağıdaki komutları çalıştırın:

```bash
npm install
npm run dev
```

4. Tarayıcıda `http://localhost:3000` adresini açın.

## Telefonda açma

- Telefon ve bilgisayar aynı Wi-Fi'da olsun.
- Bilgisayarın yerel IP adresiyle uygulamayı açın: `http://IP-ADRESI:3000`
- iPhone'da Safari üzerinden paylaş > Ana Ekrana Ekle.

## Manuel test listesi

- Oda seçildiğinde hasta listesi değişmeli.
- Liste görünümünde Mobilize Et ve Oda Değiştir çalışmalı.
- Sedye görünümünde Sol/Sağ görünmeli.
- Mobilize alandaki hasta Sedyeye Al ile odaya dönebilmeli.
- Hedef sedye doluysa yer değiştirme olmalı.
- Taburcu olan hasta arşiv detayında önceki bilgileri göstermeli.
- Sayfa yenilense bile veriler korunmalı.
